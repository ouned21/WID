# Checklist déménagement complète : tout ce qu'il faut penser

**Mot-clé :** organisation déménagement checklist, tâches après déménagement
**Intention :** solution pratique + conversion pack premium

---

## 3 mois avant (J-90)

- [ ] Comparer les devis déménageurs (minimum 3)
- [ ] Trier et désencombrer chaque pièce (donner, vendre, jeter)
- [ ] Organiser le mode de garde pour le jour J (si enfants)
- [ ] Prévenir le propriétaire (préavis)

## 1 mois avant (J-30)

- [ ] Résilier les contrats énergie et internet
- [ ] Souscrire les nouveaux contrats (nouvelle adresse)
- [ ] Changement d'adresse La Poste (réexpédition courrier)
- [ ] Acheter le matériel d'emballage (cartons, scotch, papier bulle)
- [ ] Prévenir l'école / crèche / nounou

## 1 semaine avant (J-7)

- [ ] Faire les cartons par pièce (étiqueter chaque carton)
- [ ] Emballer les objets fragiles (vaisselle, cadres, électronique)
- [ ] Vider et dégivrer le congélateur
- [ ] Programmer le suivi de colis / livraisons

## Le jour J

- [ ] Relever les compteurs (eau, gaz, électricité)
- [ ] Faire l'état des lieux de sortie (photos !)
- [ ] Nettoyer l'ancien logement
- [ ] Faire l'état des lieux d'entrée (noter les défauts)

## La semaine après (J+7)

- [ ] Déballer et ranger pièce par pièce
- [ ] Mettre à jour la carte grise (ANTS)
- [ ] S'inscrire sur les listes électorales
- [ ] Mettre à jour l'adresse partout : banque, sécu, impôts, mutuelle, employeur
