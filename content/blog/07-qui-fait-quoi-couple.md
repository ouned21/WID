# "Qui fait quoi ?" — L'exercice que chaque couple devrait faire

**Mot-clé :** qui fait quoi couple, répartition tâches couple
**Intention :** solution pratique

---

## L'exercice en 20 minutes

Prenez une feuille. Tracez 3 colonnes : "Moi", "L'autre", "Personne/Les deux". Listez TOUTES les tâches de votre foyer — pas seulement le ménage.

### Les tâches visibles
Cuisine, vaisselle, aspirateur, linge, courses, poubelles, rangement, salle de bain, jardin.

### Les tâches invisibles (celles qu'on oublie)
Prendre les RDV médicaux, gérer les factures, planifier les repas, organiser les vacances, penser aux cadeaux, vérifier les stocks, gérer les assurances, suivre les devoirs, organiser les activités, renouveler les papiers.

### Le résultat
Dans 80% des cas, une des colonnes est nettement plus longue. Ce n'est pas un reproche — c'est un constat.

## L'étape suivante : peser, pas compter

Maintenant ajoutez un score de 1 à 10 pour chaque tâche : temps + stress + conséquences. "Sortir les poubelles" (1/10) ≠ "Gérer les impôts" (8/10).

Faites le total. La répartition en nombre de tâches peut être 50/50 alors que le poids est 70/30.
