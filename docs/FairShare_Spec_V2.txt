================================================================================
  1. VISION PRODUIT
================================================================================

  1.1 Mission
  ──────────
  FairShare est la première application qui mesure le poids réel des tâches
  domestiques — pas juste le temps passé, mais la charge mentale, l'effort
  physique, l'impact sur le foyer, et la fréquence. Elle rend visible
  l'invisible pour permettre aux couples et familles de rééquilibrer leur
  quotidien.

  1.2 Promesse
  ──────────
  « Tu gères tout dans ta tête ? On va le rendre visible. »

  1.3 Cible principale
  ──────────
  • Femmes 25-45 ans en couple avec ou sans enfants
  • Sensibles à la notion de charge mentale
  • Utilisatrices mobiles (iOS/Android, web-first en V1)
  • Niveau tech : utilisatrices d'apps courantes (Instagram, WhatsApp)

  1.4 Différenciation
  ──────────
  • Algorithme de scoring multi-axes unique (temps + physique + mental
    6 dimensions + impact)
  • Onboarding révolutionnaire par équipements et enfants
  • Assignation par swipe (style Tinder)
  • Mode « membre fantôme » pour utiliser l'app seul(e)
  • Génération automatique de sous-tâches
  • IA intégrée (Claude API) pour suggestions proactives

  2.3 Algorithme de scoring
  ──────────
  4 axes :
  • Temps (1-8) : durée estimée
  • Physique (0-5) : aucun, léger, moyen, intense
  • Charge mentale (0-18) : 6 sous-dimensions
    - Anticipation (0-3)
    - Conséquence si oublié (0-3)
    - Interruption (0-3)
    - Charge décisionnelle (0-3)
    - Rigidité horaire (0-3)
    - Poids de responsabilité (0-3)
  • Impact foyer (1-4)

  Formule : (T×1.0) + (P×0.8) + (M×1.5) + (I×1.0)
  Normalisé sur 36, affiché sur 10.

  Score dual :
  • user_score (0-10) = choix utilisateur via slider → affichage individuel
  • global_score (0-36) = algo → comparaisons entre membres

  3. FONCTIONNALITÉS
================================================================================

  3.1 Onboarding révolutionnaire ★ KILLER FEATURE
  ──────────
  Écran 1 — « Ton logement »
  Grille de 29 équipements cliquables en 7 catégories (cuisine, salle de
  bain, linge, sols, extérieur, véhicule, animaux). Chaque sélection
  génère automatiquement les tâches d'entretien associées.

  Écran 2 — « Ta famille »
  • 0-2 ans → tâches bébé (couches, biberons, bain, pédiatre...)
  • 3-5 ans → tâches petit (école, goûter, habiller...)
  • 6-12 ans → tâches enfant (devoirs, activités, cartable...)
  • 13+ ans → tâches ado (résultats scolaires, orientation...)

  Écran 3 — « C'est prêt »
  Résumé : "On a créé X tâches pour ton foyer"

  Écran 4 — Assignation par swipe
  Pile de cartes style Tinder. Swipe gauche → membre A.
  Swipe droite → membre B. Skip → pool non assigné.

  3.2 Création de tâche
  ──────────
  • Nom avec autocomplétion (93+ templates)
  • Catégorie auto-détectée par mots-clés
  • Durée, effort physique, fréquence, assignation
  • Score : slider 0-10 pré-rempli par l'algo, ajustable
  • Options avancées : fixe/variable, rappels on/off
  • Après validation : suggestions de sous-tâches automatiques

  3.3 FAB + (Floating Action Button)
  ──────────
  Bouton rond bleu "+" permanent sur toutes les pages. Création rapide
  avec brouillons sauvegardés en localStorage. Rappels aux moments
  stratégiques (midi, 17h, 21h).

  3.4 Liste des tâches
  ──────────
  Cartes avec score /10, jauges Mental et Temps, sections temporelles
  (En retard, Aujourd'hui, Demain, Semaine, Plus tard). Bouton FAIT
  + bouton fantôme pour compléter au nom d'un membre sans compte.

  3.5 Récap du soir
  ──────────
  4 sections cochables :
  1. « Tes tâches » — assignées, dues aujourd'hui ou en retard
  2. « Aussi fait aujourd'hui ? » — tâches du foyer
  3. « Suggestions » — templates non encore dans le foyer
  4. « Autre chose... » — champ texte libre
  Batch completion en 15 secondes. Notification push à 21h.

  3.6 Planning / Calendrier
  ──────────
  Remplace Boost dans la navigation principale.
  • Vue semaine : colonnes Lun-Dim, blocs colorés par catégorie
  • Vue mois : grille avec points colorés par jour
  • Score total par jour, jours surchargés en rouge
  • Création de tâche depuis le planning

  3.7 Membre fantôme
  ──────────
  Membre sans compte. Ajout depuis le profil (prénom uniquement).
  Assignable aux tâches, complétable "au nom de". Présent dans les
  stats. Exclu des échanges. Rattachable à un vrai compte plus tard.

  3.8 Dashboard
  ──────────
  3 styles (Vivid/Dark/Clean). Score /10, tendance, stats, équilibre
  foyer, top 3 priorités. Lien récap du soir après 17h.

  3.9 Échanges de tâches
  ──────────
  Moteur de suggestions. Swap uniquement sur les tâches variables.
  Fantômes exclus des propositions.

  3.10 Rééquilibrage automatique
  ──────────
  Détection des pics de charge (jour > 2x la moyenne). Proposition
  de décalage des tâches variables vers des jours plus légers.

  3.11 IA intégrée
  ──────────
  ┌──────────────────────┬──────────────────────────────────┬──────┐
  │ Fonction             │ Description                       │ Prio │
  ├──────────────────────┼──────────────────────────────────┼──────┤
  │ Sous-tâches          │ "Anniversaire Léa" → 5-8 tâches  │ P1   │
  │ Suggestions soir     │ Analyse habitudes → personnalisé  │ P1   │
  │ Catégorisation       │ Comprend le contexte              │ P2   │
  │ Rééquilibrage        │ Optimise le planning              │ P2   │
  │ Résumé hebdo         │ "Barbara a porté 68%..."          │ P3   │
  └──────────────────────┴──────────────────────────────────┴──────┘

  6. ÉTUDE DE MARCHÉ
================================================================================

  6.1 Marché
  ──────────
  • Marché apps tâches domestiques : ~500M$ en 2025, croissance 15%/an
  • 71% de la charge mentale portée par les mères (Univ. Bath, 2025)
  • 63% des femmes estiment faire plus que leur part vs 22% des hommes
  • Un système structuré via app réduit les disputes de 60% en 3 mois

  6.2 Concurrence
  ──────────
  ┌────────────────────┬──────────────────────┬──────────────────────┐
  │ App                │ Force                │ Faiblesse            │
  ├────────────────────┼──────────────────────┼──────────────────────┤
  │ Sweepy (2M+ DL)    │ IA prédictive        │ Ménage uniquement    │
  │ Nipto (500K DL)    │ Gamification         │ Buggy, basique       │
  │ Cozi (10M+ DL)     │ Très installé        │ Pas de scoring       │
  │ FairChore          │ Gratuit              │ Pas de multi-axes    │
  │ EvenUS             │ Finances + tâches    │ Trop large           │
  │ Cupla (1M DL)      │ -75% stress          │ Pas de rééquilibrage │
  └────────────────────┴──────────────────────┴──────────────────────┘

  6.3 Avantages concurrentiels FairShare
  1. Seul algo de scoring multi-axes (6 dimensions)
  2. Onboarding par équipements (personne ne le fait)
  3. Swipe d'assignation
  4. Membre fantôme (usage solo)
  5. Sous-tâches automatiques
  6. IA intégrée

  8. DESIGN SYSTEM
================================================================================

  Principes :
  • iOS-first : cartes blanches, coins arrondis, ombres légères
  • Touch-friendly : minimum 44x44px
  • Ton empathique, pas gamifié
  • Design neutre, pas genré

  Couleurs :
  ┌────────────┬──────────┬──────────────────────────────────┐
  │ Token      │ Valeur   │ Usage                             │
  ├────────────┼──────────┼──────────────────────────────────┤
  │ blue       │ #007AFF  │ Accent principal, liens, boutons  │
  │ green      │ #34C759  │ Succès, scores bas, bouton FAIT   │
  │ orange     │ #FF9500  │ Scores moyens, alertes            │
  │ red        │ #FF3B30  │ Scores élevés, erreurs            │
  │ bg         │ #F6F8FF  │ Fond de page                      │
  │ text       │ #1C1C1E  │ Texte principal                   │
  │ textMuted  │ #8E8E93  │ Texte secondaire                  │
  └────────────┴──────────┴──────────────────────────────────┘

  Typographie : 40px hero, 28px titre, 18px section, 15px body,
  12px caption, 10px micro.

  Espacement : système 8px (xs=8, sm=12, md=16, lg=24, xl=32).
